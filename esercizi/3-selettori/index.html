Selettori

Selettore semplice

CSS è case-sensitive

Selettori di tipo (type selector)

h1 {color: red;}

Selettori di tipo (attribute selector)

a[href^="http://"] {color: green;}
a[href$=".it"] {color: cyan;}
a[href*="bing"] {color: violet;}

  1. https://www.google.it
  2. https://www.bing.com
  3. http://it.yahoo.com/

Selettori di classe (class selector)

.selector {border-bottom: 1px solid gray;}

Tutti gli elementi con classe .selector hanno un bordo inferiore definito.

Selettori di ID (ID attribute)

#main-container {margin: 20px;}

Il selettore di ID identifica in modo univoco un elemento della pagina.
È semanticamente errato avere più elementi con lo stesso ID nella stessa pagina.

Pseudoclassi (pseudo–class selector)

a:hover {color: blue; background-color: lightblue;}

Le pseudoclassi definiscono uno stato dell'elemento.

Compound selector

#compound-container {background-color: lightcoral;} #compound-container > div {padding: 20px; width: 50%;} #compound-container .about {background-color: cyan;}

Un selettore composto (Compound selector) è una sequenza di selettori semplici
